# 简易版_计算机组成原理

计算机分为`硬件`和`软件`

## 硬件和软件

硬件
- 主机(里面有 CPU，磁盘，等等)
- 显示器
- 鼠标键盘

软件
- 由程序员开发的，比如浏览器，比如QQ 微信 等等

## 程序存储

前面我们说到，一开始的计算机是由手转动齿轮的，那个时候  程序是在电路板里面固定的，比如你现在的电脑，设计了只能上QQ 不能上微信，成本很大！一种计算就得制造一个新的机器

后面发展成了`通用计算机`，程序存放在`磁盘`里，电脑可以运行多种程序，（导弹计算也可以，天气预测也可以，看视频也可以）


在这里面涉及到了以下新的知识（不学，听一下就好）
- 信息编码技术(比如之前听过的UTF-8,GBK编码)  就是一种`规则`，  我们在前面的举例中 也规定了一种编码  `00000001代表字母A`
- 电路编程技术（怎么样用电来表示我们要的0000001,00000010.....）
- 编程语言技术,跟电路编程技术原理一样,怎么样把代码 转换成跟纸条打出来的二进制一样的程序, 比如 可以把我们的中文 翻译成很多的`二进制(只有0和1)`  保存起来  
- - 假设： "牛逼" 两个字，实际存储的是:`010001101010100101010101011100000101011`
